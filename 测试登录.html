🔧 故障排除

问题 1: 页面无法访问

如果无法访问应用，可能是以下原因：

  • 应用未部署到 GitHub Pages
  • 网络连接问题
  • GitHub Pages 仍在处理中

解决方案：

  1. 确认已按照部署指南上传所有文件
  2. 等待 1-3 分钟让 GitHub Pages 处理完成
  3. 检查浏览器控制台是否有错误信息

问题 2: 登录失败

如果登录失败，可能是以下原因：

  • 账号密码不正确
  • JavaScript 执行错误
  • 浏览器兼容性问题

解决方案：

  1. 确保输入正确的账号密码（admin / 123456）
  2. 按 F12 打开开发者工具，检查控制台是否有错误
  3. 尝试使用 Chrome、Edge 或 Safari 浏览器
  4. 清除浏览器缓存后重试
